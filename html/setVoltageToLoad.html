setVoltageToLoad

setVoltageToLoad sets a given power supply's out put to a given voltage in 10 linearly spaced steps. The function also uses a given oscilloscope to verify that the voltage is increasing across the device. If there is a discrepancy greater than 5% between the device and the power supply output the function will throw an error. From a given settings the function will either perform all actions automatically or will prompt the user to manually change the voltage.
