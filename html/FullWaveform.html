FullWaveform

The FullWaveform class is used to store the complete waveform. It also contains functionality to split the waveform into the two switching events.

sectionWaveform

Method
Arguments (String) switchCapture
Output (SwitchWaveform) sectionedWF
Description Sections either the turn on or the turn off waveform from the base waveform as specified by the switchCapture argument. It is recommended to use SwitchWaveforms TURN_ON and TURN_OFF properties for the argument. This method is called by the extractSwitches method.
