runDoublePulseTest

Runs a single iteration of the Double Pulse test. Before running this function the bus should already be at the appropriate voltage.

runDoublePulseTest

Method
Arguments (SCPI_Oscilloscope) myScope, (SCPI_FunctionGenerator) myFGen, (Float) loadCurrent, (Float) busVoltage, (DPTSettings) settings
Output (FullWaveform) returnWaveforms
Description Runs a single operating point of the double pulse test. Sets up oscilloscope and function generator, triggers the function generator, ensures that a waveform was captured, and creates the FullWaveform object to be returned.
